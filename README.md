# 🐾 Simulador de Animal de Estimação Virtual

Projeto desenvolvido em Java com o objetivo de aplicar os conceitos fundamentais de Programação Orientada a Objetos (POO) por meio da criação de um simulador de animal de estimação virtual.

Neste projeto, foi implementado um sistema capaz de representar um animal virtual utilizando objetos, atributos e métodos, simulando interações e comportamentos inspirados em situações do mundo real.

---

## 🎯 Objetivo

Aplicar na prática os conceitos de Programação Orientada a Objetos, desenvolvendo uma aplicação simples e interativa que simula os cuidados com um animal de estimação virtual.

---

## 📚 Conceitos Utilizados

* Programação Orientada a Objetos (POO)
* Classes e Objetos
* Atributos
* Métodos
* Encapsulamento
* Lógica de Programação
* Estruturas de Controle

---

## ☕ Tecnologias Utilizadas

* Java
* JDK
* IntelliJ IDEA / Eclipse / VS Code

---

## 🚀 Como Executar

1. Clone o repositório:

```bash
git clone <url-do-repositorio>
```

2. Abra o projeto em uma IDE compatível com Java.

3. Compile e execute a aplicação.
